Project 5 : Airline Reservation Flask Web App

May 10, 10105 | 1 minute read
Share this: Twitter Facebook Pinterest

Tags: flask, SQLAlchemy, MySQL database

This project consisted of building a flask web app for an airline reservation system. The app is able to handle different signup and login for customers, booking agents, and airline staffs. The app also has different use cases for these three users.

Customer Use Cases

  • signup/login
  • search for flights using the airport code and city name
  • purchase flights
  • view personal spending
  • view past and upcoming flights

Booking Agent Use Cases

  • signup/login
  • search for flights
  • purchase flights for a customer
  • view their commission for a specific range of time
  • view their top customers by commission earned and tickets sold

Airline Staff Use Cases

  • signup/login
  • searh for & view past and upcoming flights for their airline
  • view top customers and agents
  • view distribution of sales (direct vs indirect)
  • add new planes, airports, & flights
  • update flight status
  • view top destintations

Anonymous User Use Cases

  • search for flights
